# Ford-Fulkerson

Implementazione dell'algoritmo di Ford Fulkerson per i problemi di flusso.

## Utilizzo
Modificare il file `adjacency.txt` che rappresenta il grafo mediante una [lista di adiacenza](https://en.wikipedia.org/wiki/Adjacency_list)

Avviare l'aaplicativo con:
```bash
docker build -t ford-fulkerson .; docker run ford-fulkerson:latest
```
